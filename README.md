# Cybersecurity Fundamentals Lab

This repos contains hands-on cybersecurity projects completed in a Linux Mint virtual machine using VirtualBox.

## Skills Demonstrated
- Linux authentication monitoring
- Log analysis and incident reporting
- Network reconnaissance with Nmap
- Network traffic analysis using Wireshark
- Understanding of system and network level events

## Environment
- Host: Windows
- VM: Linux Mint (VirtualBox)
- Tools: Bash, Nmap, Wireshark

## Projects
1. Authentication Monitoring & Incident Report
2. Network Reconnaissance
3. Network Traffic Analysis
